Μια σύγχρονη, αυτοματοποιημένη desktop εφαρμογή γραμμένη σε Python για την ταχύτατη αντιστοίχιση, επαλήθευση και οπτικοποίηση δεδομένων (μέσω Regex) μεταξύ αρχείων **Excel (.xlsx)** και **PDF**.

> **Σημείωση:** Το project αναπτύχθηκε για να αντικαταστήσει τη χρονοβόρα, χειροκίνητη διασταύρωση στοιχείων (π.χ. κωδικών εργαζομένων) παρέχοντας αποτελέσματα σε δευτερόλεπτα.

##  Κύρια Χαρακτηριστικά (Features)

*  **Σύγχρονο Γραφικό Περιβάλλον (GUI):** Φιλικό προς τον χρήστη περιβάλλον με υποστήριξη Dark/Light Mode μέσω της βιβλιοθήκης `customtkinter`.
*  **Ασύγχρονη Εκτέλεση (Multithreading):** Η εφαρμογή δεν "παγώνει" (Not Responding) κατά την επεξεργασία μεγάλων και βαριών PDF, διατηρώντας το UI πλήρως λειτουργικό.
*  **Έξυπνη Αναζήτηση με Regex:** Ακριβής εντοπισμός πολύπλοκων αλφαριθμητικών κωδικών μέσα στο αδόμητο κείμενο των PDF.
*  **Αυτόματο Highlighting:** Δυνατότητα απευθείας χρωματισμού (πράσινο γέμισμα) των κοινών κωδικών στο αρχικό αρχείο Excel.
*  **Παραγωγή Αναφορών:** Αυτόματη δημιουργία νέου αρχείου `Matches_Only.xlsx` με τα αποτελέσματα.
*  **Ζωντανή Προεπισκόπηση (Live Preview):** Ενσωματωμένος πίνακας προεπισκόπησης αποτελεσμάτων εντός της εφαρμογής, με ενδείξεις προόδου.

##  Οδηγίες Χρήσης
* Επιλέξτε το αρχείο Excel και το αρχείο PDF από τα αντίστοιχα πεδία.

* Επιλέξτε αν θέλετε να εξαχθεί νέο αρχείο με τα κοινά ή να γίνει Highlight στο αρχικό σας Excel. (Προσοχή: το αρχείο Excel πρέπει να είναι κλειστό).

* Πατήστε Εκτέλεση Ελέγχου. Η μπάρα προόδου θα σας ενημερώνει για την εξέλιξη της ανάγνωσης.

* Δείτε άμεσα τα αποτελέσματα στον πίνακα προεπισκόπησης στα δεξιά ή πατήστε το κουμπί για άνοιγμα του αρχείου αναφοράς.

* Χρησιμοποιήστε την ενότητα "Μεμονωμένη Αναζήτηση" στο κάτω μέρος για γρήγορο έλεγχο ενός μόνο κωδικού.

## Τεχνολογίες που χρησιμοποιήθηκαν
* CustomTkinter - Για το μοντέρνο User Interface.

* OpenPyXL - Για την ανάγνωση και επεξεργασία των αρχείων Excel.

* PDFPlumber - Για την αξιόπιστη εξαγωγή κειμένου από τα PDF.

* Regex (re), Threading, OS - Ενσωματωμένες βιβλιοθήκες της Python για βελτιστοποίηση αναζήτησης και απόδοσης.
